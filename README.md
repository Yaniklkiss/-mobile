# Тестирование мобильного приложения
В мобильном приложении ***** с расширением (.apk) можно создавать, редактировать и удалять задачи. Создание и редактирование задачи возможно только из букв кириллицы, длина каждой задачи должна составлять не более 25 символов . В приложении реализована функция для отметки выполненной задачи. Оформлена документация. Тестирование мобильного приложения проводилось с помощью планшета Android и эмулятора Android Studio.
